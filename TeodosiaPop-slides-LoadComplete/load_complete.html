What is Load Testing?

A type of performance testing conducted to evaluate the behavior of a component or system with increasing load, e.g. numbers of parallel users and/or numbers of transactions, to determine what load can be handled by the component or system.

Pricing Overview

A Node-Locked license allows using a single instance of LoadComplete on a particular computer. Once activated, the license becomes directly bound to that computer. A Floating User license allows LoadComplete to be used by a group of users. You can install LoadComplete on any number of computers in your network and use LoadComplete on multiple computers simultaneously. The maximum number of concurrent LoadComplete instances is designated by the license. For example, a Floating User license for three LoadComplete instances allows using the maximum of three LoadComplete instances concurrently. LoadComplete licenses have different effective periods: A Subscription license is time-limited and allows using LoadComplete throughout the subscription period specified in the contract (for example, 1 year). Support services and maintenance (upgrades and new versions) are provided for the same period. When the subscription expires, you can renew it for an additional subscription period to continue using LoadComplete. A Perpetual license has no expiration date, which means that you can use LoadComplete forever. It includes 1 year of support and maintenance (upgrades and new releases). You can renew maintenance for an additional year of support services and free software updates.
